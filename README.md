# Áurea Lítera - Aplicativo de Livraria

O Áurea Lítera é um aplicativo mobile em desenvolvimento, criado com React Native utilizando Expo, com o objetivo de simular uma livraria moderna que integra a venda de livros digitais e físicos. A proposta do projeto é permitir que o usuário possa adquirir eBooks para leitura diretamente no aplicativo, além de comprar livros físicos com entrega, proporcionando uma experiência completa de consumo de conteúdo literário.


## Status do Projeto

O projeto ainda esta em desenvolvimento.
Até o momento, foi implementada a estrutura inicial da aplicação, com foco nas telas de autenticação e na base visual do aplicativo.


## Telas desenvolvidas

### Login
Tela responsável pelo acesso do usuário ao aplicativo.

Possui:
- Validação de e-mail
- Validação de senha
- Botão de acesso com estado ativo/inativo
- Links para as telas de cadastro e recuperação de senha


### Cadastro
Tela destinada à criação de novos usuários.

Possui:
- Campos de nome, e-mail, senha e confirmação de senha
- Validação completa dos campos
- Verificação de senha segura
- Feedback visual de erros


### Recuperação de senha
Tela utilizada para redefinição de acesso.

Possui:
- Campo de e-mail
- Validação de entrada
- Simulação de envio de instruções


## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- SQLite
- Node.js

- abrir o expo go
- escanear o QR code
